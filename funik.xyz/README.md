# Kari AI Console & Trainer

Добро пожаловать в персональную консоль ассистента **Kari**! Здесь собрано всё необходимое для запуска локального интерфейса и обучения твоей собственной версии **Kari Lite**.

---

## 1. Запуск локального интерфейса (Сайта)

Чтобы открыть красивый интерфейс в стиле ChatGPT, Claude и Gemini на локальном хосте:

1. Открой терминал в этой папке.
2. Запусти команду:
   ```bash
   python server.py
   ```
3. Сервер запустится на `http://localhost:8000`, и браузер откроется автоматически.
4. Убедись, что приложение подключилось к запущенной в фоне **Ollama** (статус внизу боковой панели должен гореть зеленым: `Ollama: Готов`).

---

## 2. Подготовка к обучению (Fine-Tuning)

Обучение происходит полностью локально на твоей видеокарте **RTX 3060 Ti** и занимает от 1 до 2 часов.

### Шаг 1: Установка библиотек
Поскольку твоя среда Python сейчас пуста, перед началом нужно установить библиотеки с поддержкой CUDA:

1. Установи PyTorch с поддержкой видеокарты:
   ```bash
   pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121
   ```
2. Установи библиотеки Hugging Face и инструменты для экономичного обучения (QLoRA):
   ```bash
   pip install transformers peft bitsandbytes accelerate datasets
   ```

### Шаг 2: Генерация датасета диалогов
Скрипт создаст диалоги, в которых ассистент представляется как **Kari** и отказывается раскрывать модель, на которой обучался:
```bash
python generate_dataset.py
```
Скрипт создаст файл `kari_dataset.json` на основе твоих текущих моделей в Ollama.

### Шаг 3: Запуск обучения
Когда будешь готов запустить обучение:
```bash
python train_kari.py
```
*Скрипт будет ждать нажатия клавиши Enter перед стартом, чтобы ты контролировал процесс.*

---

## 3. Защита видеокарты от перегрева и зависания системы

В скрипте обучения `train_kari.py` уже реализованы следующие защитные механизмы:

1. **Паузы для охлаждения (GpuCoolingCallback):** После каждого шага обучения видеокарта делает микропаузу в **200 миллисекунд** (`time.sleep(0.2)`). Это позволяет чипу сбрасывать пиковые частоты и не нагреваться выше нормы. Средняя нагрузка удерживается около **80%**.
2. **Минимальный размер пакета (Batch Size = 1):** Минимизирует пиковое потребление памяти (VRAM) и тепловыделение.
3. **Ограничение потоков процессора:** Обучение использует только **4 потока процессора** (вместо всех доступных). Твой компьютер не зависнет в процессе обучения, и ты сможешь комфортно пользоваться им (смотреть видео, сидеть в браузере).
4. **Рекомендация по аппаратному лимиту (Опционально):**
   Ты можешь ограничить максимальное энергопотребление видеокарты через командную строку Windows (от имени Администратора):
   ```cmd
   nvidia-smi -pl 150
   ```
   *(Установит лимит потребления RTX 3060 Ti на уровне 150 Вт, что физически исключит любой сильный нагрев).*
